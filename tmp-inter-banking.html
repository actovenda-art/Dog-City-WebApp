Arquivos CNAB

O que éCNAB é a abreviatura para Centro Nacional de Automação Bancária, que foi criado para simplificar a troca de informações entre os bancos através de arquivos de remessa e retorno bancário de forma automatizada.
Manuais
Os manuais visam, atender todas as necessidades do cliente, auxiliando com a montagem do arquivo no padrão FEBRABAN.Cada layout possui um conjunto de instruções para apresentar seus respectivos campos e funções, podendo serem mapeadas para rastrear as validações e violações de cada arquivo.
Validador automatizado
Nosso serviço de leitura de arquivos CNAB deverá realizar a validação de todos os registros do arquivo submetido.Caso seja encontrado algum erro de layout, o mesmo será informado para ciência do usuário, que por sua vez poderá corrigi-lo com o auxílio dos manuais.
